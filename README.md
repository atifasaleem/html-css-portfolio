# My Portfolio Website

This is my first HTML and CSS project created as part of my Full Stack AI Engineering Fellowship.

## 📌 Project Overview
This project is a simple multi-section personal portfolio website built using HTML and CSS. It demonstrates basic web development concepts and layout structuring.

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid

## ✨ Features
- Semantic HTML structure
- Multi-section layout (About, Skills, Projects, Contact)
- Responsive navigation bar
- Flexbox-based navigation
- CSS Grid layout
- Relative and absolute positioning
- Basic project card design
